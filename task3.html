Задание 3
Напиши скрипт для создания галлереи изображений по массиву данных. В HTML есть список ul#gallery. Используй массив объектов images для создания тегов img вложенных в li. Для создания разметки используй шаблонные строки и insertAdjacentHTML(). Все элементы галереи должны добавляться в DOM за одну операцию вставки. Добавь минимальное оформление галереи флексбоксами или гридами через css-классы.
